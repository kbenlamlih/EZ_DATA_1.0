## Installation

Version of python : 3.7
Version of numpy : 1.21.4
Version of Django : 3.2.9

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the packages required of this project.
```bash
pip install django 
pip install git
pip install numpy
pip install matplotlib 
```
